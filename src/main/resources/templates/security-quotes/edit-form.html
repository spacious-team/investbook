Дата и время указывать в вашей временной таймзоне.

Инструмент: для акций, облигаций, ETF и других инструментов, имеющих ISIN, после наименования инструмента необходимо в скобках указать ISIN, например "Яндекс (NL0009805522)". Если в выпадающем списке не появляется бумага с ISIN, то наберите часть наименования и нажмите на символ "Лупы". Бумага и ISIN подгрузятся с МосБиржи при продолжении ввода, если существует совпадение. Для фьючерсов, опционов и валютных контрактов указывается наименование контракта, например "Si-3.22" (или в кратком коде "SiH2"), "Si-3.22M170322CA80000" (или в кратком коде "Si80000BC2"), "USDRUB_TOM", придерживаясь полного наименовая или краткого кода контракта для всех сделок.

Котировка: для облигации в процентах (%), для срочных контрактов - в пунктах.

Цена: для облигации - чистая цена в валюте, для срочных контрактов опционально - в валюте.
